### 🧭 What I'm working on

- **Agent fleets** — orchestrating multiple Claude/local models as coordinated workers, not chatbots
- **Local-first AI** — self-hosted Kubernetes cluster, LiteLLM gateway, Weaviate + Graphiti memory
- **Tool surfaces for agents** — making CLIs, APIs, and infra primitives legible to LLMs (`clix`, `curio`, `toolfactory`)
- **Knowledge harnesses** — turning raw sources into curated, navigable knowledge graphs
